Macros

Setup

In the layout.html file, include the following:

{% from "./components/back-link/macro.njk" import govukBackLink %}
{% from "./components/date-input/macro.njk" import govukDateInput %}

In the view file, e.g., index.html, ensure the correct layout file is referenced.

{% extends "./layout.html" %}

Nunjucks gracefully fails, which could lead to the macro being undefined or falsey

Example 1

Call the imported govukBackLink macro:

Code

{{ govukBackLink({
	"href": "https://www.google.com",
	"text": "Back"
}) }}

Output

{{ govukBackLink({ "href": "https://www.google.com", "text": "Back" }) }}

Example 2

Call the imported govukDateInput macro:

Code

{{ govukDateInput({
	"id": "date-of-birth",
	"name": "date_of_birth",
	"fieldset": {
		"legend": {
			"text": "Date of birth"
		}
	},
	"hint": {
		"text": "For example, 31 3 1980"
	},
	"items": [
		{
			"name": "day"
		},
		{
			"name": "month"
		},
		{
			"name": "year"
		}
	]
}) }}

Output

{{ govukDateInput({ "id": "date-of-birth", "name": "date_of_birth", "fieldset": { "legend": { "text": "Date of birth" } }, "hint": { "text": "For example, 31 3 1980" }, "items": [ { "name": "day" }, { "name": "month" }, { "name": "year" } ] }) }}
